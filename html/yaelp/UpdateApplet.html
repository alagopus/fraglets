Update YAELP Roster

This applets needs a recent java runtime to work. If you need one, fetch it from Sun Microsystems. If you get lost on that site, seek J2SE, Downloads, a recent version (1.4.1 for instance) and use the JRE column on the download page. Check this screenshot to check that it works as intended.

You will be asked to trust this applet. That is because this applet needs file access to your local computer to read log files and requires an existing EverQuest(TM) installation. See the official EverQuest site for more information about everquest.

You can use this applet as an application as well, just download wizard-xxx.jar from the project download page. Double-clicking on the wizard-xxx.jar should open a window if you have a recent Java runtime engine installed.
