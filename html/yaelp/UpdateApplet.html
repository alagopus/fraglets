Update YAELP Roster

This applets needs a recent java runtime to work. If you need one, fetch it from Sun Microsystems.

This applet needs file access to your local computer and requires an existing EverQuest(TM) installation. See the official EverQuest site for more information about everquest.

You can use this applet as an application as well, just download wizard-1.23.jar, sax.jar and xp.jar and put them in a folder of your choice. Double-clicking on the wizard-1.23.jar should open a window if you have a recent Java runtime engine installed.
